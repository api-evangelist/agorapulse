# Agorapulse (agorapulse)

Agorapulse is a social media management and CRM platform offering a REST API for publishing content, managing inboxes, monitoring social listening, and accessing performance analytics across Facebook, Instagram, LinkedIn, YouTube, TikTok, and Threads.

**APIs.json:** [https://raw.githubusercontent.com/api-evangelist/agorapulse/refs/heads/main/apis.yml](https://raw.githubusercontent.com/api-evangelist/agorapulse/refs/heads/main/apis.yml)

## APIs

### Agorapulse Reports Open API

The Agorapulse Reports Open API provides read-only access to analytics and reporting data, enabling teams to export audience, community management, and content performance data to external BI tools and analytics platforms. Available exclusively on the Custom plan.

- **Human URL:** [https://support.agorapulse.com/en/articles/12417183-how-to-connect-to-agorapulse-s-open-api](https://support.agorapulse.com/en/articles/12417183-how-to-connect-to-agorapulse-s-open-api)
- **Base URL:** `https://api.agorapulse.com`
